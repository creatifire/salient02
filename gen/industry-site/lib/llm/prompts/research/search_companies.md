# Company Search Prompt

Search for companies in the {industry} industry.

## Search Criteria
- Focus on: {focus_areas}
- Company size: {company_size}
- Geographic region: {region}

## Requirements
- Return at least {min_companies} companies
- Include company name, website, and brief description
- Focus on companies with strong online presence
- Prioritize companies with clear product/service offerings

## Output Format
Return results as a structured list with:
1. Company name
2. Website URL
3. Brief description (1-2 sentences)
4. Primary products/services
