# Website Analysis Prompt

Analyze the following website content and extract key information.

## Website Information
- Company: {company_name}
- URL: {website_url}

## Content to Analyze
{website_content}

## Analysis Requirements
1. Identify main products or services
2. Extract product names and descriptions
3. Identify product categories
4. Note any pricing or features mentioned
5. Extract company value proposition

## Output Format
Provide a structured analysis with:
- Products: List of product names with brief descriptions
- Categories: High-level product categories
- Key Features: Notable features across products
- Company Focus: Primary business focus areas
